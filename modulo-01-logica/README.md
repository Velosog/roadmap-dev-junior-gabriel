# Módulo 01 — Fundamentos e Lógica de Programação

## Objetivo do módulo
Dominar lógica de programação, raciocínio algorítmico e bases de programação estruturada, para não ficar perdido quando começar a codar de verdade no back-end e no front-end.

## Tópicos principais
- Tipos de dados, variáveis e operadores
- Estruturas condicionais (if, else, switch)
- Estruturas de repetição (for, while)
- Vetores, matrizes e registros
- Funções e modularização
- Noções de algoritmos e pseudocódigo

## Cursos / materiais de estudo
- Curso de lógica de programação (Alura / outro) — DEFINIR
- Listas de exercícios e desafios
- Resumos próprios em Markdown

> Sempre que definir um curso específico da Alura, atualizar aqui com nome e link.

## Projetos / exercícios práticos
- Resolução de listas de exercícios
- Pequenos programas de console (calculadora, conversões, menus, etc.)
- Desafios de lógica (ex.: algoritmos clássicos simples)

## Critérios de “módulo concluído”
- Ser capaz de resolver exercícios de lógica sem depender de “cola”
- Entender o que o código faz antes de rodar
- Conseguir explicar em voz alta como um algoritmo funciona
