# Módulo 04 — Bancos de Dados

## Objetivo do módulo
Aprender a modelar e manipular dados em banco relacional, integrando com o back-end.

## Tópicos principais
- Conceitos de tabelas, registros, chaves primárias e estrangeiras
- Modelagem simples de banco (DER básico)
- SQL básico (SELECT, INSERT, UPDATE, DELETE)
- Integração back-end + banco (ORM ou driver nativo)

## Cursos / materiais de estudo
- Curso de SQL / Banco de Dados — DEFINIR (Alura)
- Ferramenta de visualização (DBeaver, Beekeeper, etc.)

## Projetos práticos
- Banco com 2–4 tabelas relacionadas (ex.: usuários, pedidos, produtos)
- API integrada ao banco (persistência real)
- Consultas com filtros e ordenação

## Critérios de “módulo concluído”
- Saber montar consultas simples sem depender de tutorial
- Entender relações 1:N e N:N
- Ter 1 projeto com persistência real funcionando
