# Módulo 03 — Back-end básico

## Objetivo do módulo
Começar a desenvolver APIs e aplicações de back-end simples, entendendo rotas, requisições e respostas.

## Tópicos principais
- HTTP (requisições, respostas, status codes)
- Conceito de API REST
- Introdução a uma stack de back-end (ex.: Node.js ou Python/FastAPI)
- Rotas, controllers, serviços
- Manipulação de dados em memória

## Cursos / materiais de estudo
- Curso de Node.js ou FastAPI — DEFINIR (Alura)
- Documentação oficial da tecnologia escolhida
- Projetos guiados

## Projetos práticos
- API de cadastro de itens em memória (produtos, tarefas, etc.)
- CRUD simples (Create, Read, Update, Delete)
- Integração com front-end simples ou cliente tipo Insomnia/Postman

## Critérios de “módulo concluído”
- Conseguir criar uma API simples com rotas REST
- Entender como testar rotas
- Ter pelo menos 1 projeto de API rodando documentado

## Rotina de atualização
- Registrar APIs criadas
- Documentar endpoints básicos (rota, método, descrição)
