# Módulo 02 — Fundamentos de Front-end (HTML, CSS e JS básico)

## Objetivo do módulo
Aprender a construir páginas web estruturadas, com estilo básico e interatividade simples em JavaScript.

## Tópicos principais
- HTML semântico (tags, hierarquia, acessibilidade básica)
- CSS (box model, flexbox, grid, responsividade básica)
- JavaScript no navegador (DOM, eventos, funções)
- Boas práticas de organização de arquivos (HTML/CSS/JS separados)

## Cursos / materiais de estudo
- Curso de HTML e CSS — DEFINIR (Alura)
- Curso de JavaScript para iniciantes — DEFINIR (Alura)
- Projetos guiados / documentação MDN

## Projetos práticos
- Landing page simples responsiva
- Página com formulário validado em JS
- Mini-app com interação (ex.: lista de tarefas, contador, etc.)

## Critérios de “módulo concluído”
- Entender como estruturar uma página HTML do zero
- Ser capaz de criar um layout simples responsivo
- Manipular elementos da página com JavaScript (click, input, etc.)
