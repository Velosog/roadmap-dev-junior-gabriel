# Módulo 05 — Projetos Práticos e Portfólio

## Objetivo do módulo
Consolidar tudo o que foi aprendido nos módulos anteriores em projetos mais completos, que possam ir para o portfólio e currículo.

## Diretrizes gerais
- Projetos completos (front + back + banco sempre que possível)
- Documentação caprichada em cada repositório
- Foco em resolver problemas reais/similares a demandas de empresa

## Ideias de projetos
- Sistema simples de controle de tarefas ou estudos
- Mini-CRM de clientes
- Sistema de cadastro de produtos e pedidos
- Projeto voltado para empresa real (ex.: RexFort, Aurora, etc.)

## Critérios de “módulo concluído”
- Ter 2–3 projetos completos documentados
- Ter ao menos 1 projeto que você sente confiança em apresentar em entrevista
- README de cada projeto bem escrito (objetivo, tech stack, como rodar, etc.)

## Rotina de atualização
- Registrar novos projetos com:
  - Link do repositório
  - Resumo do que o projeto faz
  - Tecnologias usadas
